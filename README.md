# Sneakers Project 👟

Projeto desenvolvido a partir do desafio Ecommerce Product Page do Frontend Mentor, com o objetivo de praticar conceitos de HTML, CSS e JavaScript na construção de interfaces modernas e responsivas.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## 🎯 Objetivos do Projeto

- Estruturação semântica com HTML
- Estilização moderna e responsiva com CSS
- Manipulação do DOM com JavaScript
- Boas práticas de organização de código
- Controle de versão com Git

## 📱 Responsividade

O projeto foi desenvolvido para funcionar em diferentes tamanhos de tela:

- Desktop
- Tablet
- Mobile
